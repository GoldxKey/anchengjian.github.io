传参的时候传函数一般是用对象，也不怕污染，我这次突发奇想。传参的时候直接给了函数名，然后在函数内加个扩号开始执行
```JavaScript
//声明第一个���数0
function test_fn0(){
    alert(&quot;ok&quot;);
}

//声明第二个函数1
function test_fn1(obj){
    obj();
}

//第二个函数调用第一个函数
test_fn1(test_fn0); 
```

然后这样子竟然也就执行了。。。
